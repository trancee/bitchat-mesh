## iOS

### Command

```bash
cd ios
swift test
swift test --enable-code-coverage
xcrun llvm-cov report \
  .build/debug/BitchatMeshPackageTests.xctest/Contents/MacOS/BitchatMeshPackageTests \
  -instr-profile .build/debug/codecov/default.profdata
```
